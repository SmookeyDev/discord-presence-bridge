# Compatible Extensions

Third-party extensions that work with Discord Presence Bridge.

---

## Built-in Providers

These providers are included in Discord Presence Bridge:

| Provider | Features |
|----------|----------|
| YouTube | Video title, channel, progress, live detection |
| GeoGuessr | Game mode, round, score, map name |

---

## Third-party Extensions

Extensions that integrate via the [External API](./api.md):

### PreWrap

PreMiD Presences wrapper with over 500 presences.

| Platform | Link |
|----------|------|
| Chrome | [Chrome Web Store](https://chrome.google.com/webstore/detail/prewrap/calpcokkjmookodfpbmdbknfcjhekgaj) |
| Firefox | [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/prewrap/) |
| Source | [GitHub](https://github.com/lolamtisch/PreWrap) |

### MAL-Sync

Integrates MyAnimeList into various sites with auto episode tracking.

| Platform | Link |
|----------|------|
| Chrome | [Chrome Web Store](https://chrome.google.com/webstore/detail/mal-sync/kekjfbackdeiabghhcdklcdoekaanoel) |
| Firefox | [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/mal-sync/) |
| Source | [GitHub](https://github.com/MALSync/MALSync) |

---

## Add Your Extension

Want to add your extension to this list?

1. Implement the [External API](./api.md)
2. Open a [Pull Request](https://github.com/SmookeyDev/discord-presence-bridge/pulls) adding your extension

Or create a [built-in provider](../README.md#adding-providers) for tighter integration.
